## :technologist: Tecnologias

Este projeto faz uso das seguintes tecnologias:
- [Webpack](https://webpack.js.org/)
- [Redux](https://redux.js.org/)

## 💻 Projeto

O Redux-Starter é uma simples aplicação para testar funcionalidades básicas do Redux.

## 🚀 Como executar

- Clone o repositório
- Instale as dependências com `yarn install`
- Inicie o servidor com `yarn start`
- Crie types, actions, reducers e teste através da index

O servidor deverá iniciar no endereço: [http://localhost:9000](http://localhost:9000)
